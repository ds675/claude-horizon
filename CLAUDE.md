# Horizon Theme

Shopify theme built on [Horizon](https://github.com/Shopify/horizon).
Custom sections and snippets extend Horizon — core files are never modified.

## Stack
- Liquid, Custom CSS, Vanilla JS
- No Tailwind, no JS frameworks, no build step

## Rules index
- `.claude/conventions.md` — naming, file structure
- `.claude/liquid.md` — Liquid patterns
- `.claude/css.md` — CSS architecture
- `.claude/javascript.md` — JS patterns
- `.claude/schema.md` — schema conventions
- `.claude/workflow.md` — how to approach every task

## Non-negotiables
- Never modify Horizon core files
- Always check existing sections and snippets before creating new ones
- CSS variables bridge schema settings to styles — never hardcode values
